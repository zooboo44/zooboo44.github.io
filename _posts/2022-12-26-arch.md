# Install

## Initial Configuration for AwesomeWM
### Install Drivers
- nvidia nvidia-utils nvidia-settings xorg-server-devel opencl-nvidia nvidia-prime xorg-server xorg-xinit xf86-video-intel mesa lib32-mesa

### Install and configure awesome
- Install awesome
```
sudo pacman -S awesome
```
- Copy xinitrc
```
sudo cp /etc/X11/xinit/xinitrc ~/.xinitrc
```
- Remove lines past ```twm &``` and replace with ```exec awesome```

- Copy awesome profile to home directory
```
sudo cp /etc/xdg/awesome/ ~/.config/awesome
```
###  Other Packages to Install
- Git ```sudo pacman -S git```
- Base-devel ```sudo pacman -S base-devel```

# Find battery level
```
cat /sys/class/power_supply/BAT0/capacity
```

# Catch 22
- ERROR: a GPU setup was initiated but Xorg post-start hook did not run
- Fix
    ```
    prime-offload
    optimus-manager --switch integrated
    ```
